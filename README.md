# portfolio-sample — YEOBAEK STUDIO (디자인 스튜디오 포트폴리오)

이피웹(YIPI WEB) 포트폴리오용 ⑤-3번 샘플. 브랜딩·그래픽 디자인 스튜디오 자체 포폴.

- **매트릭스 셀:** 포트폴리오 · **좌측 고정 인덱스 사이드바** · **필터형 케이스 그리드** · **케이스 상세 뷰(클릭→풀 케이스 스터디 전환)** · **필름 미니멀**
- **차별점(셀렉트 3종 중 3번):** QUINZE(게이트·숫자인덱스·브루탈리즘)·VESSEL(매거진 그리드·소형 라이트박스·그레이지)과 모두 다름 → 좌측 사이드바 2-pane, 풀 케이스 상세 전환, 웜 오프화이트+잉크+슬레이트, **Space Mono 라벨**. 그리드 썸네일은 약한 그레이스케일→호버/상세 시 풀컬러(필름 톤).
- **정직 라벨:** 커머스 없음(포폴). CTA=프로젝트 문의 폼 + Behance·인스타. 모바일 하단바=이메일·Behance·문의.
- **앵커 보정:** `scroll-padding-top` 적용(모바일 상단바).
